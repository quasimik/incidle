# Incidle

A Wordle-style incident triage game. You get paged, you investigate — each clue you burn (or wrong hypothesis you make) reveals more evidence. Name the root cause before the clue budget runs out, or the incident escalates.

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build   # outputs to dist/
```

Built with React + Vite. Deployed on Vercel.
